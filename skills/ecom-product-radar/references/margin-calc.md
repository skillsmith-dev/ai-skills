# 电商选品毛利率速算表

## 常见平台佣金参考
| 平台 | 佣金率 | 备注 |
|------|--------|------|
| 抖店（普通商品） | 2-5% | 不同类目差异大 |
| 抖店（服饰） | 5-10% | 含退换货成本 |
| 淘宝C店 | 0% | 无平台佣金，但有直通车成本 |
| 天猫 | 2-5% | +年费 |
| 拼多多 | 0.6% | 极低佣金但价格战激烈 |
| 小红书电商 | 5-10% | 新渠道，流量成本暂低 |

## 毛利率计算公式
```
毛利率 = (零售价 - 1688拿货价 - 运费 - 包装费 - 平台佣金) / 零售价 × 100%
```

## 运费参考
| 重量区间 | 快递均价 |
|----------|---------|
| 0-500g | 3-5元 |
| 500g-1kg | 5-8元 |
| 1-3kg | 8-12元 |
| 3kg+ | 按续重1-3元/kg |

## 利润等级判定
- 高利润：≥40%（可投入流量获客）
- 中等利润：30-40%（需精细化运营）
- 低利润：20-30%（仅适合走量）
- 亏损线：<20%（不建议入场）

## 选品避坑特征
- 退货率>15%的品类（服饰鞋包典型）
- 季节性强但周期<2个月的品类
- 需要资质/许可证但无法快速办理的品类
- 1688同款>50页供应商的品类（极度红海）
- 客单价<15元且运费占比>30%的品类
