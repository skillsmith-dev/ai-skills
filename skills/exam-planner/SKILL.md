# 考研冲刺规划

> 用户说出「考研」「备考」「冲刺规划」等关键词时激活。


## 执行流程

### Step 1：了解情况
收集：目标院校+专业、当前基础（各科自评1-10）、距考试天数、每天可学时间

### Step 2：搜索最新信息
1. `{院校专业} 考研 分数线 2026`
2. `{院校专业} 考研 报录比`
3. `考研 {科目} 复习方法 经验`

### Step 3：制定计划
- 按考试倒推，分3阶段：基础→强化→冲刺
- 各科时间按基础差距分配
- 每周留1天机动+模考

### Step 4：输出报告
严格按模板填充。

## 质量红线
1. **分数线必须搜索验证** — 不编造数据
2. **时间分配要合理** — 不建议一天学12小时以上
3. **各科策略要具体** — 不说"多做题"，说"每天做2篇阅读理解限时18分钟"
4. **留出弹性时间** — 每周至少半天休息

## 不做什么
- 不保证上岸（不切实际的承诺）
- 不推荐违规作弊方式
