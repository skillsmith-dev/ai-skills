# 短视频脚本生成器

> 用户说出「短视频脚本」「视频脚本」「抖音脚本」等关键词时激活。

## 输出模板

```
## 🎬 短视频脚本

**时长**: {xx}秒 | **平台**: {抖音/快手/小红书} | **类型**: {口播/剧情/教程}

### 分镜表
| 时间 | 画面 | 口播/字幕 | 音乐/音效 | 备注 |
|------|------|----------|----------|------|
| 0-3s | {开场画面} | {钩子台词} | {音效} | 3秒留存关键 |
| 3-10s | {画面} | {内容} | {BGM} | |
| ... | ... | ... | ... | |
| 最后3s | {CTA画面} | {引导关注/点赞} | {音效} | |

### 🎯 关键设计
- **钩子**: {3秒钩子是什么}
- **高潮点**: {xx秒处的刺激点}
- **CTA**: {结尾引导做什么}

### 📝 拍摄提示
- {需要准备的道具/场景}
- {拍摄技巧建议}

### 📊 完播率优化
- {节奏建议}
- {每10秒的刺激点安排}
```

## 执行流程

### Step 1：了解需求
确认：视频主题、时长、平台、类型（口播/剧情/教程/混剪）

### Step 2：搜索参考
1. `{主题} 短视频 脚本 模板`
2. `{主题} 抖音 爆款 视频`

### Step 3：设计脚本
核心节奏：
- 0-3s：钩子（反差/悬念/痛点）
- 3-10s：展开（给出价值/故事开头）
- 每10s一个小刺激（转折/数据/冲突）
- 最后3s：CTA

### Step 4：输出

## 质量红线
1. **3秒钩子必须有** — 没钩子的视频直接划走
2. **每10秒一个刺激点** — 防止中途流失
3. **分镜要具体** — 写清画面内容，不是"拍产品"
4. **CTA要明确** — "关注看下集"比"谢谢观看"有效100倍
