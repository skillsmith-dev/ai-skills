# 读书笔记生成器

> 用户说出「读书笔记」「书摘」「读后感」等关键词时激活。

## 输出模板

```
## 📚 读书笔记：《{书名}》

### 🎯 一句话精华
{这本书最核心的一句话insight}

### 📖 核心观点（3-5个）
1. **{观点1}**: {2-3句话解释}
2. **{观点2}**: {2-3句话解释}

### 🏗️ 论证结构
{观点1} → {论据} → {观点2} → {论据} → {结论}

### ✨ 金句摘录
1. "{原文金句1}" — {解读}
2. "{原文金句2}" — {解读}

### 🤔 批判性思考
- **赞同**: {什么观点同意+为什么}
- **质疑**: {什么观点有保留+为什么}

### ✅ 行动清单
| # | 行动 | 来源章节 | 优先级 |
|---|------|---------|--------|
| 1 | {具体可执行的行动} | {章节} | ⭐⭐⭐ |
| 2 | {具体可执行的行动} | {章节} | ⭐⭐ |

### 🔗 延伸阅读
- {相关的书1}: {为什么相关}
- {相关的书2}: {为什么相关}
```

## 执行流程

### Step 1：确认书籍
搜索：`{书名} 书评 核心观点`

### Step 2：提炼结构
- 核心论点
- 支撑论据
- 推理链条

### Step 3：生成笔记
按模板填充，特别注意行动清单

### Step 4：输出

## 质量红线
1. **核心观点不超过5个** — 太多等于没有重点
2. **金句要原文** — 不自己编造
3. **行动清单必须可执行** — 不写"学会XX"，写"明天开始每天XX"
4. **要有批判性思考** — 不一味吹捧
