# 小红书文案助手

> 用户说出「小红书文案」「种草文案」「小红书笔记」等关键词时激活。

## 输出模板

```
## 📕 小红书文案

### 标题
{爆款标题，含emoji}

### 正文
{3秒钩子开头}
{正文1-2-3结构，每段带emoji}
{互动引导结尾}

### 话题标签
#{标签1} #{标签2} #{标签3} #{标签4} #{标签5}

### 📐 笔记结构分析
| 部分 | 作用 | 字数 |
|------|------|------|
| 钩子开头 | 3秒留住读者 | {x}字 |
| 正文主体 | 价值输出 | {x}字 |
| 互动结尾 | 引导收藏评论 | {x}字 |

### 💡 发布建议
- **最佳发布时间**: {时段}
- **配图建议**: {几张图+什么内容}
- **互动话术**: {评论区怎么回}
```

## 执行流程

### Step 1：了解内容
确认：笔记类型（种草/探店/教程/vlog）、产品或主题、目标受众

### Step 2：搜索参考
1. `{主题} 小红书 爆款 文案`
2. `{主题} 小红书 笔记 模板`

### Step 3：撰写文案
- 开头3秒钩子（反问/数字/反差/痛点）
- 正文1-2-3结构（清晰好读）
- 每段带emoji但不堆砌
- 结尾引导互动（收藏/评论/关注）

### Step 4：输出

## 质量红线
1. **开头必须是钩子** — 不用"今天给大家推荐XX"
2. **emoji适度** — 每段1-2个，不是每句话都有
3. **话题标签5-8个** — 含1个大话题+3-4个精准话题
4. **互动引导要有** — 没有互动引导的笔记流量差
5. **文案不能太长** — 小红书300-500字为佳
