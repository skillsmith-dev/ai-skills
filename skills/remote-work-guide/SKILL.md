# 远程办公指南

> 用户说出「远程办公」「居家办公」「在家办公」等关键词时激活。

## 输出模板

```
## 🏠 远程办公优化方案

### 🔍 痛点诊断
- **你的主要问题**: {从描述中提炼}
- **影响程度**: 🟢轻微 / 🟡中等 / 🔴严重

### ⏰ 时间管理方案
| 时间段 | 活动 | 技巧 |
|--------|------|------|
| 9:00-12:00 | 深度工作 | {番茄钟/专注模式} |
| 12:00-13:30 | 休息 | {避免刷手机} |
| 13:30-17:00 | 协作沟通 | {集中处理消息} |

### 🛠️ 工具推荐
| 需求 | 推荐工具 | 替代方案 |
|------|---------|---------|
| 沟通 | {工具} | {替代} |
| 协作 | {工具} | {替代} |
| 文档 | {工具} | {替代} |

### 💬 沟通最佳实践
- **同步沟通**: {什么时候用即时消息}
- **异步沟通**: {什么时候用文档/邮件}
- **会议效率**: {如何减少无效会议}

### 🎯 自我管理技巧
1. {技巧1}: {具体做法}
2. {技巧2}: {具体做法}

### ⚠️ 远程办公坑
- ❌ {坑1}: {怎么避免}
- ❌ {坑2}: {怎么避免}
```

## 执行流程

### Step 1：了解痛点
确认：角色（个人/管理者）、痛点（效率低/沟通难/孤独感/工作生活不分）

### Step 2：搜索验证
1. `远程办公 效率 方法 2026`
2. `远程协作 工具 对比`

### Step 3：定制方案
根据具体痛点给出针对性建议

### Step 4：输出

## 质量红线
1. **方案要具体可执行** — 不说"提高效率"，说"用番茄钟25分钟专注+5分钟休息"
2. **工具推荐要中立** — 不只推荐一个平台
3. **时间管理要可定制** — 不同人作息不同
4. **要包含心理健康** — 远程办公的孤独感是真实问题
