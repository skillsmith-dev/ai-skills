## 🛠️ 快速安装

### Claude Code / Codex CLI

```bash
# 克隆仓库
git clone https://github.com/skillsmith-dev/ai-skills.git
cd ai-skills

# 链接技能目录（以 trending-topic-radar 为例）
ln -s ./skills/trending-topic-radar ~/.claude/skills/
# 或 Codex CLI
ln -s ./skills/trending-topic-radar ~/.codex/skills/
```

### OpenClaw / ClawHub

```bash
# 通过 OpenClaw 安装
claw skill install skillsmith-dev/ai-skills/trending-topic-radar

# 或浏览 ClawHub 安装
# 👉 https://clawhub.example.com/skillsmith-dev/ai-skills
```

### Coze / 扣子

```bash
# 使用 Coze CLI
coze skills add trending-topic-radar --source ./skills/trending-topic-radar

# 或在 Coze 平台导入技能文件夹
# 👉 https://www.coze.cn/store/market
```

> 💡 **提示**：不同的 Agent 客户端安装路径略有差异，请参考 [Coze 官方文档](https://docs.coze.cn) 或对应平台的安装指南。

## 🤝 贡献指南

欢迎提交新技能！请遵循以下流程：

1. **Fork** 本仓库
2. 创建一个以技能命名的目录：`skills/your-skill-name/`
3. 参照现有技能结构添加 `SKILL.md` 和相关资源
4. 在本 README.md 的技能表格中添加对应条目
5. 提交 Pull Request，我会尽快审核

### 技能结构规范

```
skills/
└── your-skill-name/
    ├── SKILL.md          # 技能主文件（含描述、触发词、使用说明）
    ├── scripts/          # 可选：辅助脚本
    └── references/       # 可选：参考文档/模板
```

> ⚠️ 提交前请确保技能已通过基本测试，描述清晰且有实际使用价值。
